About

ANX Camera is a Miui Camera Port for Xiaomi's Devices on AOSP-Based ROMS. It's generally ported from Latest Miui ROMS of newer devices. You need to download and install the zip via Magisk or TWRP, there is no APK to install

Legend

  • All -> Works on all supported devices of that time
  • Singularity -> Works on all supported devices of that time
  • OOS -> Oxygen OS, Doesn't work on AOSP-Based ROMS
  • Device CodeName works on that device mostly. except Cepheus
  • If FileName contains Magisk -> Flash with Magisk Manager
  • If FileName contains Unity -> Flash with Magisk Manager or TWRP

Installation

  1. Download the zip from Downloads section
  2. Flash the downloaded .zip in Magisk Manager or TWRP. Make sure to read whatever it prints during installation
  3. Check Addons to get addons. some might be required for your device. Flash via Magisk or TWRP as mentioned
  4. Reboot, Enjoy

Uninstallation

  1. Flash the same .zip in Magisk Manager or TWRP again to uninstall
  2. Or remove via Magisk Manager Modules
  3. Reboot, Don't Enjoy
