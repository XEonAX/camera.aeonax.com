About

ANXCamera is a Miui Camera Port for Xiaomi Devices using AOSP based ROMs. It is generally ported from Latest Miui ROMS and new, feature-rich devices. You need to download and install the .zip via Magisk or TWRP, there is no APK to install

Legend

  • All -> Works on all supported devices of that time
  • Singularity -> Works on all supported devices of that time
  • OOS -> Oxygen OS, Doesn't work on AOSP-Based ROMS
  • Device CodeName works on that device mostly. except Cepheus
  • If the Filename contains 'Magisk' --> Flash with Magisk Manager
  • If the Filename contains 'Unity' --> Flash with Magisk Manager or TWRP

Installation

  1. Download the .zip from the Downloads section
  2. Flash the downloaded .zip in Magisk Manager or TWRP. Make sure to read whatever the Installation Screen displays
  3. Check Addons to get addons which might be required for your device. Flash via Magisk or TWRP as mentioned above
  4. Reboot, Enjoy!

Uninstallation

  1. Flash the same .zip in Magisk Manager or TWRP again to uninstall
  2. Or remove via Magisk Manager Modules
  3. Reboot, Don't Enjoy!

Addons enable additional features or improve compatibility.
Mandatory Addons:
'ANXCustlibs' for fixing portrait mode for few devices,
'48MP Fix' for devices with/without 48MP, Includes MiuiLibs from Mi 9T [davinci], fixes portrait as well as low-res photos,
Flash only one of the above not both, Don't Flash on OOS. If you have queries ask here

'Custom Vendor Fix' for some devices with custom vendor,
