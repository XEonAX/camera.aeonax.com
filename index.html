Supported Devices

Check if your device belongs to following:
  • beryllium a.k.a. poco
  • dipper a.k.a. mi8
  • polaris a.k.a. mix2s
  • violet a.k.a. rn7p
  • perseus a.k.a. mix3
  • cepheus a.k.a. mi9
  • raphael a.k.a. k20p
  • davinci a.k.a. mi9t/k20
  • lavender a.k.a. rn7
  • equuleus a.k.a. mi8p
  • platina a.k.a. mi8l
  • whyred a.k.a. rn5/p
  • tulip a.k.a. rn6p
  • gemini a.k.a. mi5
  • grus a.k.a. mi9se
  • sirius a.k.a. mi8se
  • pyxis a.k.a. cc9
  • laurus a.k.a. cc9e
In decreasing order of support.
Your ro.product.device must be one of codenames above. So please ensure this on GSIs.

Currently there are 2 Devs
  • AEonAX - Owns a Poco
  • Luffitys - Owns a Mi8
Both make Builds with different features and based on different Bases. Check the Telegram Group for more info.

Legend

  • Singularity -> Works on all supported devices
  • OOS -> Oxygen OS, Doesn't work on AOSP-Based ROMS
  • oosberyllium -> Oxygen OS for POCO only
  • OOS Singularity -> Oxygen OS Only, Will work on POCO, Might Work on other supported devices
  • Device CodeName works on that device mostly
  • If FileName contains Magisk -> Flash with Magisk Manager
  • If FileName contains Unity -> Flash with Magisk Manager or TWRP

Installation

  1. Download the zip from Downloads section
  2. Flash the downloaded .zip in Magisk Manager or TWRP
  3. Check Addons to get addons. some might be required for your device. Flash via Magisk or TWRP as mentioned
  4. Reboot, Enjoy
